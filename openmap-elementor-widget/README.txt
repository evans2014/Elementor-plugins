==================================================================
DESCRIPTION
==================================================================

OpenMap for Elementor is a **lightweight, free, and powerful** Elementor widget that lets you display **multiple locations** on a **fully customizable OpenStreetMap** – **without any API key**.

Perfect for:
- Store locators
- Office branches
- Event maps
- Real estate
- Travel guides

No Google Maps API. No limits. No cost.

==================================================================
KEY FEATURES
==================================================================

- Unlimited locations (repeater field)
- Custom marker icon per location
- 7 beautiful map styles (Light, Dark, Satellite, Topo, etc.)
- Marker clustering (+3, +10) when zoomed out
- Responsive & mobile-ready
- Works in Elementor editor (preview)
- No API key required
- Fast loading (Leaflet.js + CDN)
- Auto-fit all markers on load
- WYSIWYG popup content
- Full control: height, zoom, center

==================================================================
INSTALLATION
==================================================================

1. Download the plugin as `.zip`
2. Go to WordPress Admin → Plugins → Add New → Upload Plugin
3. Upload `openmap-elementor-widget.zip`
4. Click "Install Now" → Activate
5. Make sure **Elementor (Free)** is installed and active

The widget will appear in Elementor under **General** category as:
→ **OpenMap – Clustering + Styles**

==================================================================
HOW TO USE
==================================================================

1. Open any page in **Elementor**
2. Search for **"OpenMap"** in the widget panel
3. Drag & drop it onto the page
4. Configure:
   - Map height & zoom
   - Center coordinates (lat/lng)
   - Choose map style
   - Add locations (address, lat/lng, popup, icon)
5. Save & preview

Tip: Use [latlong.net](https://www.latlong.net/) to get coordinates.

==================================================================
MAP STYLES INCLUDED
==================================================================

- Standard (OSM)
- Light (CartoDB Positron)
- Dark (CartoDB Dark Matter)
- Topographic (OpenTopoMap)
- Cycling (CyclOSM)
- Satellite (Esri World Imagery)
- Grayscale (HOT Humanitarian)

All styles work without API keys.

==================================================================
CLUSTERING
==================================================================

When multiple markers are close:
- They group into a circle with a number (e.g., +5)
- Click or zoom to expand
- Fully customizable (via code)

==================================================================
FREQUENTLY ASKED QUESTIONS
==================================================================

Q: Do I need a Google Maps API key?
A: No! This uses OpenStreetMap – 100% free.

Q: Can I use my own marker icons?
A: Yes! Upload any PNG/SVG in the repeater.

Q: Why is the map blank in Elementor editor?
A: Normal. You see a preview. Full map appears on frontend.

Q: Can I import locations from CSV?
A: Not yet – but planned for v2.0!
