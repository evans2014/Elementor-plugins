A lightweight custom Elementor Info Box widget with presets, image/icon support and advanced button styling.

== Description ==

Ebox Info Box is a custom Elementor widget that allows you to build flexible info boxes without relying on third-party addons like UAE.

Features include:

* Preset selector (Preset 1–5)
* Icon or Image support
* Icon + Title on the same row
* Description displayed below (full width)
* Fully customizable button
* Button supports SVG and Font Icons
* Button typography (font family, size, weight)
* Separate Normal and Hover styles
* Clean, lightweight and extendable code

Perfect for feature boxes, service blocks, call-to-action sections and more.

== Installation ==

1. Upload the `ebox-info-box` folder to the `/wp-content/plugins/` directory, or upload the ZIP via WordPress Plugins screen.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Edit a page with Elementor.
4. Search for **Ebox Info Box** widget and drag it into your layout.

== Usage ==

1. Add the **Ebox Info Box** widget in Elementor.
2. Choose a preset (Preset 1–5).
3. Select Media Type (Icon or Image).
4. Set Title and Description.
5. Configure the Button text, link and icon.
6. Style everything using the Style tabs (Media, Title, Description, Button).

== Button Features ==

* Uses `<a>` tag (full clickable area)
* Icon position before or after text
* Typography control (font family, size, weight)
* Separate Normal and Hover color settings
* No unwanted hover animations or resizing

== Frequently Asked Questions ==

= Does this plugin require Elementor? =
Yes. Elementor must be installed and activated.

= Does it depend on UAE or other addons? =
No. This plugin is fully standalone.

= Can I use SVG icons? =
Yes. SVG and Font Icons are supported.

= Are presets visually different by default? =
Presets add CSS classes and can be styled individually via CSS.
